Standard 8: Fundamental Concepts of Web Design and Development

Students will be able to describe, demonstrate and evaluate fundamental elements front-end and back-end development.

Exposed Competent Proficient Advanced
Student can recall the basic idea of the standard Student can explain the standard correctly. Student can apply or demonstrate the standard correctly Students can evaluate the strengths and weakness of the standard and conclude when to use it

8.a Research and identify components of the current Hyper Text Markup Language (HTML) standard for identification, processing, and presentation of information on web pages.

8.b Research and identify current front-end tool kits used in industry and their role in web development.

8.c Research and identify current web standards bodies.

8.d Demonstrate creation of a static web page using the current HTML standard.

8.e Demonstrate the use of the current Cascading Style Sheets (CSS) standard for style selectors and declarations including id, class, nested selectors, multiple selectors, and pseudo selectors.

8.f Demonstrate the manipulation of elements in a web page using the box model.

8.g Demonstrate various element positioning methods, such as the CSS position property.

8.h Research and identify current web page layout models.

8.i Demonstrate a responsive web page using a fluid design and media queries.

8.j Identify page loading efficiency best practices.

8.k Demonstrate implementing a web page using a front-end toolkit.
